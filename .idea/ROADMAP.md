# 🎵 Plateforme de Gestion Musicale (Projet Secret)

## Modules principaux :
1. **The Vault (Audio)** : Gestion de versions (Mixup/Untitled), Lecteur Hi-Fi.
2. **Cue & Feedback** : Commentaires temporels sur la waveform pour ingés/artistes.
3. **Studio Chat** : Messagerie type Discord avec canaux privés par rôle.
4. **Timeline** : Calendrier de sorties et planification studio.
5. **Royalties Hub** : Gestion des split sheets et contrats.
6. **The Gallery** : Assets graphiques et système de vote pour les covers.
7. **Stage Prep** : Gestion des BPO et fiches techniques live.

## Stack Technique (Low Cost) :
- Framework : Next.js
- Base de données/Auth/Storage : Supabase
- Hébergement : Vercel
- UI : Tailwind CSS + shadcn/ui
